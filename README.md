# expressts-starter

Opinionated template for NodeJS backend with ExpressJS, TypeScript, TailwindCSS,
 and Webpack

## Features

- [x] ExpressJS
- [ ] TypeScript
- [ ] TailwindCSS
- [ ] Webpack
- [ ] Jest
- [ ] Docker
- [ ] Postgres
- [ ] CI/CD
- [ ] Github Actions
- [ ] Render

## Getting Started

```bash
git clone
cd expressts-starter
npm install
npm run dev
```
